# PoC for Commodore Repair Toolbox

**Proof-of-Concept** and only for **internal use**!

Not sure project will ever be realized... let's see.

## Requirements

- .NET 10, https://dotnet.microsoft.com/en-us/download/dotnet/10.0
- Windows 10 build 1607 or newer (64-bit)
- macOS 14 (Sonoma) or newer (64-bit)
- Linux (64-bit):
  - glibc 2.17+
  - OpenSSL 1.1+
  - libicu
  - Desktop:
    - X11
    - Wayland
