# PoC for Commodore Repair Toolbox

**Proof-of-Concept** and only for **internal use**!

Not sure project will ever be realized, as core functionalities must be running before any conclusion can be made - let's see...

## Requirements

- .NET 10, https://dotnet.microsoft.com/en-us/download/dotnet/10.0
- 64-bit operating system:
  - **Windows** 10 build 1607 or newer
  - **macOS** 14 (Sonoma) or newer
  - **Linux**:
    - glibc 2.17+
    - OpenSSL 1.1+
    - libicu
    - Desktop:
      - X11
      - Wayland
